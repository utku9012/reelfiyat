This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

ReelFiyat, enflasyonist ortamlarda "vade farksız" taksitli alışverişlerin gerçek maliyetini ve tüketiciye sağladığı finansal avantajı hesaplayan modern bir web uygulamasıdır.

Paranın zaman değerini (Time Value of Money) baz alarak, taksitli ödemelerin bugünkü reel karşılığını ve nakit parayı yatırımda değerlendirmenin getirisini kullanıcıya sunar.

Reel Fiyat Analizi: Enflasyon oranına göre her bir taksitin bugünkü alım gücü karşılığını hesaplar.

Fırsat Maliyeti (PPF) Hesaplama: Nakit parayı Para Piyasası Fonu'nda (PPF) değerlendirdiğinizde elde edeceğiniz toplam faiz getirisini gösterir.
